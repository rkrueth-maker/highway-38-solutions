# Shop / CNC Concept Sample

Purpose: show how a shop or machine process idea becomes a concept packet.

Customer sends:
- process photos
- short video or notes
- part description
- cycle-time guess
- pain point

Output:
- current state summary
- concept direction
- missing data list
- vendor questions
- boundary notes
- next decision path

Boundary: concept planning only. Final safety, guarding, controls, programming, and integration stay with qualified vendors.
